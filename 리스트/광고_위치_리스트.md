# 2. 광고 나오는 조건
상단 배너 : 레벨 선택씬, 상점 씬, 미션 씬, 돼지 저금통 구매 씬,  
코인 50개 광고 : 상점씬 닫으면 나옴  
전면광고 : 스테이지 클리어 ,로비로 이동, 재시작
